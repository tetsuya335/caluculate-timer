# 🕒 ルラップ・リンカー

作業時間を1タップで記録し、GAS（Google Apps Script）経由でスプレッドシートへ自動保存するタイムトラッキングツールです。

## ✨ 主な機能
- **簡単操作**：START / STOP の1タップで作業時間を計測
- **スマートな仕分け**：過去の入力履歴からワンタップで作業内容を自動入力
- **状況に合わせた4色カラーリング**：
  - 未分類：アンバー（黄色）
  - 計測中：ブルー（青色）
  - 正常終了：エメラルド（緑色）
  - 時間超過：ローズ（赤色）

## 🛠️ 使い方
1. GASのWebアプリURLを入力します。
2. 作業を開始するときに「START」を押します。
